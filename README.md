# Workblox [in-development]
> The all-in-one workspace for work and study.

Workblox is an intended open-source project designed for modern students, professional, and executives solutions. Its approach will combining features for automations, knowledge and organization managment in a single platform.

# Problems to solve
- Users manage multiple emails across different providers
- Professionals keep switching between multiple messaging platforms
- Students have fragmented knowledge managment within multiple Spaced Repetition and Notes taking apps

### MVP Features
> This MVP will ensure a correct solution for students and professionals.
- Notes with inline editing
- Centralized email inbox with categories
- Spaced repetition system with integrated memo cards

### Planned features
> Future iterations will expand Workblox towards advanced professional and executive workflows.
- Social media and DM integrations
- Calendars and scheduling
- Internal Workflow automation
- Unified communication inbox
- Team collaboration features
